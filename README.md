# SEOUL TRIP
#### 웹으로 서울을 탐방하는 프로젝트      
###### 진행기간 : 2022.11.01 ~ 2022.11.30
###### SKILLS/IDE : Java,JSP,HTML,CSS / Spring Framework, MyBatis / MySQL / Eclipse
###### 프로젝트 개발 참여 파트  
##### Back-end  
###### - 로그인 및 회원가입  
###### - 회원정보 수정,탈퇴  
###### - 게시판 기능 구현(작성,조회,상세정보,검색,삭제)  
###### - 회원목록 조회  
###### - 데이터베이스 쿼리작성  
##### Front-end  
###### - 메인페이지 kakao map api 연동  
###### - 로그인,회원가입,회원정보 디자인  
###### - 관광지 디자인 및 상세정보 추가
